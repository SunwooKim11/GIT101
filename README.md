# GIT101

## 내 사진 추가하는 법

자신의 정보를 넣고 아래 코드를 복사하여 table 태그 안에 추가합니다.
commit message는
```
Add: <본인 하고 싶은 말>
``` 

```html
<tr>
    <td height="140px" align="center"> <a href="https://github.com/bentshrimp"><img src="https://github.com/bentshrimp.png"/><br/>내 소개<br/>내 이름</a></td>
</tr>
```
